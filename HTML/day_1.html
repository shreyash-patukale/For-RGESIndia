Hello! I am a computer science student passionate about AI, machine learning, and data science. I am currently working on an automated aeroponics system that integrates real-time monitoring and machine learning to optimize plant growth.

My Aeroponics Project

My aeroponics project is focused on using sensors to collect data such as pH, TDS, temperature, and humidity to manage plant nutrition precisely. I use an ESP32 microcontroller to process real-time data and a Random Forest Regression model to predict and adjust nutrient levels.

Key Features

  • Real-time sensor data collection
  • Automated pH and nutrient adjustments
  • Machine learning for predictive analysis
  • Edge computing with ESP32 for efficient processing

Future Goals

I aim to refine my model further and expand its capabilities to create a fully automated, AI-driven aeroponics system that enhances crop yield and efficiency.
